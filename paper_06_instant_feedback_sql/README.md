# paper_06_instant_feedback_sql

Data and analysis for the paper “Instant Feedback for SQL Assignments using Clause-Driven Automated Grading”.

## Folder structure

```text
paper_06_instant_feedback_sql/
  README.md
  software/
    socoles/
  data/
    raw/
    processed/
  scripts/
  outputs/
    figures/
    tables/
```

## Dataset groups

- `data/raw/`: source files for the classroom instant-feedback study
- `data/processed/`: derived datasets for the study
- `software/`: study-related software snapshots
- `scripts/`: analysis and data-preparation code
- `outputs/figures/`: generated plots and visual summaries
- `outputs/tables/`: generated tables
