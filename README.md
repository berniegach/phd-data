# phd-data

This repository contains datasets, analysis scripts, and research outputs associated with the papers that form my PhD thesis on automated SQL grading and feedback generation in computer science education.

Where possible, the data and analysis materials have been made publicly available to support transparency, reproducibility, and reuse.


- Thesis URL: [pending](#)

## Repository purpose

The research presented in the thesis draws on a combination of:

- historical student SQL submissions
- grading outcomes generated by the SOCOLES tool
- survey responses
- derived analysis results, figures, and summary outputs

Each study is organized in its own top-level folder so that data, scripts, and outputs can be documented and reused independently.

## Repository organization

Each paper folder typically follows this structure:

```text
paper_XX_short_name/
	README.md
	data/
		raw/
		processed/
	scripts/
	outputs/
		figures/
		tables/
```

In general:

- `data/raw/` contains original or source datasets preserved as collected
- `data/processed/` contains cleaned, transformed, or scored datasets
- `scripts/` contains analysis and visualization code
- `outputs/` contains generated figures and tables

## Privacy and ethics

To protect student privacy, personally identifying information was removed before data were used for analysis or shared in this repository.

The public datasets are therefore limited to the information necessary to understand, reproduce, and verify the analyses reported in the thesis.

Some materials may remain unavailable where ethical or institutional constraints prevent open publication.
