# paper_01_sql_grading_sensitivity

Data and analysis for the paper “Sensitivity of Automated SQL Grading in Computer Science Courses”.

## Folder structure

```text
paper_01_sql_grading_sensitivity/
  README.md
  data/
    raw/
      submissions/
      reference_answers/
    processed/
      graded_scale_08/
      graded_scale_27/
  scripts/
    analyze_graded_results.py
  outputs/
    figures/
  metadata/
```

## Dataset groups

- `data/raw/submissions/`: student SQL submissions
- `data/raw/reference_answers/`: reference SQL answers
- `data/processed/graded_scale_08/`: grading outputs for the 8-level scheme
- `data/processed/graded_scale_27/`: grading outputs for the 27-level scheme
- `scripts/`: analysis code
- `outputs/figures/`: generated plots
- `metadata/`: supporting documentation
